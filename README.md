Simple, stateless React components.

## Install

```shell
$ yarn add react-pasta
```

## Usage

```jsx
import React from 'react'
import { Card, Text } from 'react-pasta'

const App = () => (
  <Card>
    <Heading level={1}>Big Heading</Heading>
    <Text>This is some text.</Text>
  </Card>
)

export default App
```
